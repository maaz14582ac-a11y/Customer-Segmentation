# Customer Segmentation Using K-Means

## Objective
Cluster mall customers based on their spending behavior using unsupervised learning.

## Dataset
Mall Customers Dataset

## Techniques Used
- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Elbow Method
- PCA for visualization

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Results
Customers were segmented into 5 clusters based on annual income and spending score, enabling targeted marketing strategies.
